# 🎓 DailyExamDashboard

**DailyExamDashboard** 是一個專為現代化考場與教學空間設計的**高精度時程顯示系統**。介面採用**數位極簡主義**風格，結合大尺寸等寬數字顯示與自動化考程調度，旨在消除視覺干擾，提供最純粹的時間管理體驗。

## ✨ 核心特色

- **高對比數位顯示**：
    - 採用專業級**等寬字體**，確保秒數跳動時文字水平位置固定，不產生視覺晃動。
    - 深邃藍墨色調配置，在各類投影設備下皆具備極佳的視覺穿透力。
- **在地化民國曆支援**：
    - 專為台灣教育現場設計，完整呈現民國紀年格式（如：**民國 114 年 2 月 6 日**），完美對齊校園公文與考場公告規範。
- **智慧型動態導航**：
    - **狀態感知**：系統實時偵測當前考程進度。
    - **視覺聚焦**：進行中科目會觸發「平滑視窗居中」與「動態比例縮放」，確保資訊首要性。
- **沉浸式全螢幕配置**：
    - 針對 16:9 劇院級比例優化，一鍵切換至無干擾的專業顯示模式，隱藏瀏覽器工具列與分頁。
- **強大資料矩陣管理**：
    - **點對點編輯**：支援即時修改單一項目，無須重新載入網頁。
    - **文本批量引擎**：支援透過純文字矩陣快速遷移、備份或大量匯入考程資料。

## 🚀 快速部署

1. **啟動**：使用瀏覽器開啟 `index.html`。
2. **配置**：點擊右上方導覽列的 ⚙️ 設定選單。
3. **批量作業**：選取「批量處理」，按以下格式填寫後匯入：
   `[時間範圍],[科目名稱],[詳細備註]`
4. **演示**：點擊「進入全螢幕」，即可呈現最純淨的監考畫面。

## 🛠 技術架構

- **UI Engine**: Tailwind CSS
- **Typography**: Roboto Mono & Noto Sans TC
- **Icons**: Material Symbols
- **Data Persistence**: 瀏覽器內建 LocalStorage API（資料保留於本地裝置）
