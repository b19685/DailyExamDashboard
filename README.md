# 🎓 Today's Exam Dashboard (今日考程動態儀表板)

這是一個專為現代化考場與教學空間設計的**高精度時程顯示系統**。介面採用**數位極簡主義 (Digital Minimalism)** 風格，結合大尺寸等寬數字顯示與自動化考程調度，旨在消除視覺干擾，提供最純粹的時間管理體驗。



## ✨ 核心特色

- **高對比數位顯示 (Hi-Contrast Display)**：
    - 採用專業級**等寬字體 (Monospace Typography)**，確保秒數跳動時文字水平位置絕對固定。
    - 深邃藍墨色調配置，在各類投影設備下皆具備極佳的視覺穿透力。
- **在地化曆法支援**：
    - 內建自動日期換算邏輯，即時呈現民國紀年與標準年月日。
- **智慧型動態導航 (Smart Auto-Focus)**：
    - **狀態感知**：系統實時偵測當前考程進度。
    - **視覺聚焦**：進行中科目會觸發「視窗居中」與「比例縮放」動畫，確保資訊首要性。
- **沉浸式全螢幕配置**：
    - 針對 16:9 劇院級比例優化，一鍵切換至無干擾的專業顯示模式。
- **強大資料矩陣管理**：
    - **點對點編輯**：支援即時修改單一項目，無須重新載入。
    - **文本批量引擎**：支援透過純文字矩陣快速遷移、備份或大量匯入考程資料。

2. **配置**：點擊右上方導覽列的 ⚙️ 設定選單。
